C++ Software Engineer for WebKit

The Adobe WebEngine team is looking for a passionate and talented C++ developer to help us shape the future of the web. As part of this effort we are contributing to the WebKit open source project and to web standards, in collaboration with Google, Apple, Microsoft and other organizations. Some of the features that we’ve proposed and that we’re implementing now in WebKit: CSS Regions, CSS Custom Filters, CSS Exclusions.

In the WebEngine team you will:

  • Develop features that ship in well-known web browsers (Chrome, Safari)
  • Make yourself known in the open source community
  • Talk about your work at web conferences

Responsibilities

  • work with the WebKit open source community to implement new features (and also fix bugs)
  • work directly with the standards people to define and refine new features for the web
  • give feedback on the specs as you work on implementing them
  • work closely with the quality engineers to ensure that we really solve people’s needs
  • contribute to making awesome demos for the new features

Skills

Must have

  • strong CS background (data structures, algorithms & low level stuff)
  • excellent knowledge of C/C++, with proved experience, ideally on both Windows and Mac (or Linux), or excellent knowledge of C and OOP and desire to learn C++
  • experience with large codebases
  • fluent in English (reading, writing and speaking)

Nice to have

  • knowledge about web technologies (HTML, CSS, JavaScript, HTTP)
  • experience with web technologies or 2D/3D graphics
  • contributions to open source projects, ideally in the web domain
  • knowledge about how a web browser works
